About MuseWear

The Division Manifesto

MuseWear is the tactile counterpart to MuseCAD—where code becomes cloth, and precision becomes touch. Founded within the Muse Systems Collective, MuseWear redefines functional apparel as intelligent interface. We build garments for the moment machines start remembering warmth.

Every capsule emerges from cross-discipline design: engineers, fabric technologists, and conceptual artists converging in the MuseLabs. Each thread is coded, every seam an algorithm. We don't make clothing; we make contact points.

Function dresses fiction. That’s the law we live by.
